Tạo filesystem cho ổ mới (nếu chưa có):

bash
Copy code
sudo mkfs.ext4 /dev/xvde
Tạo thư mục tạm để mount:

bash
Copy code
sudo mkdir -p /mnt/docker-data


####################
Bước 2: Di chuyển dữ liệu Docker
Dừng Docker trước khi di chuyển:

bash
Copy code
sudo systemctl stop docker
Di chuyển thư mục /var/lib/docker sang ổ mới:

bash
Copy code
sudo rsync -a /var/lib/docker/ /mnt/docker-data/
Đổi tên thư mục cũ (để backup nếu cần):

bash
Copy code
sudo mv /var/lib/docker /var/lib/docker.bak
Tạo symbolic link từ thư mục Docker đến ổ mới:

bash
Copy code
sudo ln -s /mnt/docker-data /var/lib/docker
##########################
Bước 3: Cấu hình ổ đĩa mới cho Docker
Thêm ổ đĩa vào file /etc/fstab để tự động mount sau khi reboot: Mở file /etc/fstab:

bash
Copy code
sudo nano /etc/fstab
Thêm dòng sau vào cuối file:

bash
Copy code
/dev/xvde /mnt/docker-data ext4 defaults 0 0
Khởi động lại Docker:

bash
Copy code
sudo systemctl start docker
Kiểm tra trạng thái Docker:

bash
Copy code
sudo systemctl status docker
#############################
Bước 4: Dọn dẹp và kiểm tra
Kiểm tra dung lượng ổ đĩa:

bash
Copy code
df -h
Xóa thư mục cũ nếu mọi thứ hoạt động bình thường:

bash
Copy code
sudo rm -rf /var/lib/docker.bak
